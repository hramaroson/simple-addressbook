# simple-adressbook
A Simple Python CLI program to manage Adress book

```console
usage: addressbook.py [-h] [-ln LASTNAME] [-fn FIRSTNAME] [-a ADDRESS] [-p PHONE] [-e EMAIL] {add}

A Simple Python CLI program to manage Adress book

positional arguments:
  {add}                 Command to execute

optional arguments:
  -h, --help                            show this help message and exit
  -ln LASTNAME, --lastname LASTNAME     Last name
  -fn FIRSTNAME, --firstname FIRSTNAME  First name
  -a ADDRESS, --address ADDRESS Contact Adress
  -p PHONE, --phone PHONE               Phone number
  -e EMAIL, --email EMAIL               Email address
```
